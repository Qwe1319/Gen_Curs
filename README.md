# Решение задачи развозки груза с использованием генетических алгоритмов


### Компоненты:
- **Депо**: Центральный склад продукции
- **Транспортные средства**: Парк с разной грузоподъемностью
- **Клиенты**: Покупатели с запросами на доставку, территориально распределенные

### Цель:
Минимизировать общие затраты на проезд при доставке груза всем клиентам.


### Структура проекта
- `models.py`: Определение классов для депо, транспортных средств и клиентов
- `genetic_algorithm.py`: Реализация генетического алгоритма
- `utils.py`: Вспомогательные функции (расчет расстояний, визуализация)
- `main.py`: Основной скрипт для запуска решения


## Параметры
- Размер популяции: 100
- Количество поколений: 500
- Вероятность мутации: 0.1
- Вероятность кроссовера: 0.8
